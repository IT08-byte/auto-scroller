# ⬇️ Auto Scroller

A simple Chrome extension to auto-scroll any web page — built mainly for reading manga and manhwa hands-free. Works on sites with custom scroll containers like MangaFire, Webtoon, and more.

## Features

- Custom keybinds for start and pause
- Smart scroll target detection — works on manga readers, split layouts, feeds
- Adjustable speed and overlay size
- Settings sync across all your Chrome sessions
- Draggable overlay, auto-popup option
- Zero data collection

## Installation (Developer Mode)

Since this isn't on the Chrome Web Store yet, you'll need to load it manually — it takes about 30 seconds:

1. **Download this repo** — click the green **Code** button above → **Download ZIP** → unzip it
2. Open Chrome and go to `chrome://extensions`
3. Toggle on **Developer mode** (top-right corner)
4. Click **Load unpacked** → select the unzipped folder
5. Done — the extension icon will appear in your toolbar. Click it on any page to open the overlay.

## How to Use

1. Click the extension icon to open the overlay
2. Set your scroll speed and assign keybinds for start/pause
3. Hover your mouse over the area you want to scroll
4. Press your start key — that's it
